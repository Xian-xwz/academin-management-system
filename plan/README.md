# plan / 规划目录

本目录用于存放**毕业设计相关的规划与决策材料**，与根目录下的需求/开题/技术文档、以及 `logs/开发日志` 中的问题追踪形成区分。

## 建议存放内容

| 类型 | 示例 |
|------|------|
| 阶段性计划 | 周计划、里程碑、任务拆解 |
| 架构与选型备忘 | 技术选型对比、Dify/知识库/部署方案速记 |
| 答辩/演示准备 | 演示脚本、问答提纲 |
| 个人笔记 | 会议记录、与导师沟通要点（可选） |

## 与其他目录的关系

- 根目录 `开题报告.md` `技术文档.md` 等为**正式对外文档**。
- `plan/` 为**工作过程与计划草稿**，可随项目演进增删，结题前可再整理到正式文档。

## 目录结构

```
plan/
├── README.md
├── 教务工作台_管理员功能规划.md   # 管理员端（教务工作台）功能与分期，供后期实现对照
├── 毕设阶段规划_33b76acb.plan.md
├── Dify问答测试清单.md
└── mineru知识库处理_812ac5d4.plan.md
```

需要时可按月份或主题增加子文件，如 `2026-04-迭代计划.md`。

Dify 与知识库**索引型说明**见上级目录 `docs/Dify配置说明.md`（不在 `plan/` 内重复存放全文）。
