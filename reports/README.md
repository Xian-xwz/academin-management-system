# reports

知识库构建与质量校验报告目录。

常见输出：

- `source_inventory_report.md`：MinerU 来源覆盖报告。
- `split_validation_report.md`：专业切分报告。
- `knowledge_pipeline_report.md`：流水线汇总报告。
- `field_validation_report.json`：字段级校验结果。
- `dify_upload_manifest.json`：Dify 上传文件清单。
- `dify_qa_regression_report.md`：Dify 问答回归测试清单。
- `run_metadata.json`：本次运行元信息。
